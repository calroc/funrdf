NewLoopWord

This command takes all the items on the stack and puts them into a tuple with the Loop Handler function in front of them. A Loop consumes the top item on the stack, then depending on it's "truth" either runs the commands in its tuple and repeats if it's true or stops looping altogether if it's false. Put the first command to run on the stack first, then the second, and so on, so that the last item to run is on the top of the stack when you run this command.
