NewSeqWord

This command takes all the items on the stack and puts them into a tuple with the Sequence Handler function in front of them. The items on the stack should all be commands from the dictionary, either functions or combo commands. You get these by using the "lookup" command on the names of the functions you want. Put the first command to run on the stack first, then the second, and so on, so that the last item to run is on the top of the stack when you run this command.
